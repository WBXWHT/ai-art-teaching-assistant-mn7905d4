# ai-art-teaching-assistant

一个模拟AI美术助教多智能体交互的演示项目

项目经历：我曾主导“AI美术助教”多智能体项目，负责基于LangChain的智能体编排与上下文设计。项目包含视觉分析Agent（识别儿童画作元素）和引导反馈Agent（将专业术语转化为如“太阳公公在对你笑呢”的童言）。通过优化System Prompt与记忆链路，使AI反馈与孩子认知匹配度提升35%，并设计长期记忆结构，将孩子的色彩偏好存入向量数据库，实现个性化引导，使课程互动率提升25%。

## 运行
```bash
pip install -r requirements.txt
python main.py
```
